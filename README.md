# CryptoTrendPredict

This project is designed to predict the next day's Bitcoin closing price using various technical indicators, including RSI, MACD, and Bollinger Bands.

## Requirements

Before you start, make sure you have the following installed on your local machine:

https://www.docker.com/get-started

Setup Instructions

Step 1: Clone the repository

git clone https://github.com/eduefs79/CryptoTrendPredict.git

cd CryptoTrendPredict

Step 2: Docker Setup

Ensure that Docker is installed on your local machine.

Step 3: Adjust the docker-compose.yml

Before running Docker containers, you might need to adjust some settings in the docker-compose.yml. Here are some things to check:</p>
     Ensure that the volumes are correctly mapped to your local directories if you're working with local data or files.
        Ports: Make sure the exposed ports do not conflict with other services running on your machine.
        Environment Variables: Some environment variables may need to be configured for your local environment.
